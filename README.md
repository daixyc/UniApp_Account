# UniApp_Account
A small personal ledger

# 服务
  server文件夹是放的服务部分，使用的是Express框架 + mongo数据库，目前只完成登录和注册的基本功能，后续会继续完善。
  项目启动前，请手动node 启动服务。 node server.js
# 客户端
  客户端暂时没有什么实际功能，但是就是想做一个这个,体验一下多端部署的效果，顺便学一下UniApp，天不遂人愿，抽时间做的还是太慢了，
